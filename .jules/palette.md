# Palette's UX Journal

## 2025-05-22 - Use ContentUnavailableView for Empty States
**Learning:** Using `ContentUnavailableView` (introduced in iOS 17) provides a consistent, accessible, and visually appealing way to handle empty states in lists and detail views. It allows for a title, image, description, and actionable buttons.
**Action:** Always consider `ContentUnavailableView` when a list might be empty or when a detail view has no selection.
